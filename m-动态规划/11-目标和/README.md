## 494 目标和

给点一个非负整数数组,a1,a2,……an 和一个目标数 S。现在有两个符号+和-
对于数组中任意一个整数，都可以从+或-中选择一个符号加在前面

返回可以使最终数组和为目标数 S 的所有添加符号的方法数

示例

- 输入:nums:[1,1,1,1,1] ,S:3

解释:

- -1+1+1+1+1 = 3
- +1-1+1+1+1 = 3
- +1+1-1+1+1 = 3
- +1+1+1-1+1 = 3
- +1+1+1+1-1 = 3

一共有 5 种方法让最终目标和为 3
